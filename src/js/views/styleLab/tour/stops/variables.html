Color Variables

This section shows the color variable values in the currently selected theme. Note that these are the derived values that end up in the css. In the scss file, a variable will often point to or be derived from another variable.

Variables in the 'Palette' section can be thought of as the color palette of a theme, and other variables often point to the palette variables. Non-palette variables are (imperfectly) grouped to correspond to the categories in the components area: controls, layout and navigation, forms, collections, and visualizations.
