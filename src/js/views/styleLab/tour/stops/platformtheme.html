Choose a Target Platform

The first step in writing a custom theme is to pick the target platform theme that you want to extend. The available platform themes are:

web: alta
Use the web theme to create applications designed for the web, including phone and tablet browsers such as Google Chrome for Android and Apple Safari for iOS.
hybrid:
alta-ios,   alta-android,   alta-windows
Use hybrid themes with Apache Cordova to create hybrid mobile applications with an Alta color palette but an otherwise native application look and feel.
